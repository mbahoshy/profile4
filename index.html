portfolio

PM Manager

github.com/mbahoshy/pm_project

This is a current project of mine that addresses what I see as a gaping hole in the HVAC industry. Every day thousands of technicians collect various measurements and readings (data) about the furnaces they service. The problem is that this data is stored "flat" into either pdf's or actual paper files. What a waste!

I have created an application that allows technicians to easily record the data they are collecting in a simple and compelling way. This data is then used to give insight into system performance, view detailed system history, and provide a customer portal to monitor their own equipment.

This website is built using a node server with mongodb. The frontend renders views using bacboneJS, and the data is manipulated using underscore to create visual graphs using Google Charts.

TradePro

tradepro.herokuapp.com
github.com/mbahoshy/tt

The purpose of this application is to train HVAC (heating and cooling) technicians to identify and diagnose electrical failures. Electrical circuits are exactly simulated in the web browser and "problems" are programmed into the circuit that the user must solve.

This application tracks metrics that allow me to identify individual strengths and weaknesses of a user based on furnace type or part type (eg transformer, relay ...)

I built this application using a node server that communicates with a backbone router to render views. A navigational architecture was built to allow users to select a class, choose a topic, and view available video lessons, problems, and tests. User data and metrics are saved to a monogo database.
